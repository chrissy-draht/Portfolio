# Kassenautomat im Kino (Python)

## Beschreibung
Dieses Konsolenprojekt simuliert den Ablauf an einer Kino-Kasse.  
Der Fokus liegt auf Eingabevalidierung, Kontrollstrukturen und sauberer Berechnung von Preisen, Rabatten und Gebühren.

## Funktionen im Projekt
- Altersprüfung mit Plausibilitätskontrolle
- Ticket-Auswahl (2D / 3D)
- Snack-Auswahl
- Zahlungsart (Bar oder Karte)
- Berechnung von:
  - Ticketpreis
  - Seniorenrabatt
  - Mengenrabatt
  - Snack-Kosten
  - Kartengebühr
- Ticket-Drucksimulation mit Schleife
- Formatierte Rechnungsausgabe
- Tageszusammenfassung mit Gesamtumsatz
- Fehlerbehandlung mit `try/except` bei Benutzereingaben

## Technische Inhalte
- Python (Grundlagen)
- Schleifen (`while`, `for`)
- Bedingungen (`if`, `elif`, `else`)
- Exception Handling (`try/except`)
- Formatierte Ausgaben (`%`-Formatierung)
- Logische Operatoren
- Benutzereingaben (`input()`)

## Projekt starten

```bash
python kassenautomat_kino.py
```
